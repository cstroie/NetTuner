About NetTuner

NetTuner is an open-source ESP32-based internet radio player that allows you to stream your favorite radio stations directly from the internet with multiple control options.

Features

  • Stream internet radio stations over WiFi
  • Manage a playlist of your favorite stations
  • Control playback through a responsive web interface
  • Adjust volume and tone controls (bass, midrange, treble) with intuitive sliders
  • Support for various audio formats including MP3, AAC, and more
  • Real-time status updates via WebSocket
  • MPD (Music Player Daemon) protocol compatibility (version 0.23.0)
  • Physical controls with rotary encoder and button
  • OLED display for local status feedback
  • Multiple WiFi network configuration with automatic fallback
  • Flexible hardware configuration through web interface
  • Playlist management with JSON/M3U import/export

How to Use

  1. Connect to WiFi: Configure your WiFi networks in the WiFi settings
  2. Configure Hardware: Set up your pin assignments in the Config page
  3. Add Stations: Go to the Playlist page to add your favorite radio stations
  4. Play: Select a station from the dashboard and press Play
  5. Control: Use the volume slider, rotary encoder, or MPD client to adjust audio levels

Control Methods

NetTuner can be controlled in multiple ways:

  • Web Interface: Responsive web UI accessible from any device on the same network
  • MPD Clients: Control using any MPD-compatible client (connect to port 6600)
  • Rotary Encoder: Physical volume control and navigation
  • Physical Button: Play/pause functionality (GPIO 0 by default)
  • OLED Display: Local status feedback with scrolling text

Technical Details

NetTuner is built on the following technologies:

  • Hardware: ESP32 microcontroller with I2S audio output
  • Audio Library: ESP32-audioI2S for streaming audio decoding
  • Web Framework: Arduino ESP32 with built-in web server
  • Frontend: PicoCSS v2 for responsive UI design
  • Communication: WebSocket for real-time updates and MPD protocol on port 6600
  • Data Storage: SPIFFS for configuration and playlist storage

Project Information

NetTuner is an open-source project designed for hobbyists and makers who want to build their own internet radio player. The project is hosted on GitHub where you can find the source code, documentation, and instructions for building your own device.
